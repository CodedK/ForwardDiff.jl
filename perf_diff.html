Performing Differentiation¶

Derivatives¶

ForwardDiff.jl can take derivatives of functions/callable objects of the form f(x::Number) → Number or f(x::Number) → Array.

derivative!(output::Array, f, x::Number)

Compute \(f'(x)\), storing the output in output.

derivative(f, x::Number)¶

Compute \(f'(x)\).

derivative(f; mutates=false)

Return the function \(f'\). If mutates=false, then the returned function has the form d(x). If mutates=true, then the returned function has the form d!(output, x).

Gradients¶

Note

When calling ForwardDiff.jl’s gradient method, you must use the fully qualified method name ForwardDiff.gradient(...) in order to avoid conflict with Base.gradient.

ForwardDiff.jl can take gradients of functions/callable objects of the form f(x::Vector) → Number.

gradient!(output::Vector, f, x::Vector)

Compute \(\nabla f(\vec{x})\), storing the output in output.

ForwardDiff.gradient{S}(f, x::Vector{S})

Compute \(\nabla f(\vec{x})\), where S is the element type of both the input and output.

ForwardDiff.gradient(f; mutates=false)¶

Return the function \(\nabla f\). If mutates=false, then the returned function has the form g(x). If mutates=true, then the returned function has the form g!(output, x).

Jacobians¶

ForwardDiff.jl can take Jacobians of functions/callable objects of the form f(x:Vector) → Vector.

jacobian!(output::Matrix, f, x::Vector)

Compute \(\mathbf{J}(f)(\vec{x})\), storing the output in output.

jacobian{S}(f, x::Vector{S})

Compute \(\mathbf{J}(f)(\vec{x})\), where S is the element type of both the input and output.

jacobian(f; mutates=false)¶

Return the function \(\mathbf{J}(f)\). If mutates=false, then the returned function has the form j(x). If mutates=true, then the returned function has the form j!(output, x).

Hessians¶

ForwardDiff.jl can take Hessians of functions/callable objects of the form f(x::Vector) → Number.

hessian!(output::Matrix, f, x::Vector)

Compute \(\mathbf{H}(f)(\vec{x})\), storing the output in output.

hessian{S}(f, x::Vector{S})

Compute \(\mathbf{H}(f)(\vec{x})\), where S is the element type of both the input and output.

hessian(f; mutates=false)¶

Return the function \(\mathbf{H}(f)\). If mutates=false, then the returned function has the form h(x). If mutates=true, then the returned function has the form h!(output, x).

Tensors¶

ForwardDiff.jl can take Tensors of functions/callable objects of the form f(x::Vector) → Number.

“Tensor”, in this context, refers to a \(3^{\text{rd}}\) order generalization of the Hessian. Given a function \(f:\mathbb{R}^n \to \mathbb{R}\), the Tensor operator \(\mathbf{T}\) is defined as

\[\mathbf{T}(f) = \sum_{i,j,k=1}^{n} \frac{\delta^3 f}{\delta x_i \delta x_j \delta x_k}\]
tensor!(output::Matrix, f, x::Vector)

Compute \(\mathbf{T}(f)(\vec{x})\), storing the output in output.

tensor{S}(f, x::Vector{S})

Compute \(\mathbf{T}(f)(\vec{x})\), where S is the element type of both the input and output.

tensor(f; mutates=false)¶

Return the function \(\mathbf{T}(f)\). If mutates=false, then the returned function has the form t(x). If mutates=true, then the returned function has the form t!(output, x).

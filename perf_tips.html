Performance TipsÂ¶

Basically, one should heed the performance tips provided in the Julia docs. The most important tips listed there revolve around the notion of type stability; always try to make sure that the target function you’re passing to ForwardDiff.jl is type-stable. Type instability in the target function can cause slowness, and in some cases, errors.
